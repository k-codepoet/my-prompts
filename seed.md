# Inception Seed

> Inception 영화처럼 한 단어 (또는 짧은 구절) 을 *심으면* 거기서 세계가 자라난다.
> 시스템은 본 시드 + Constitution + Charter 만 입력으로 받아 **gen-001 의 조직과 첫 작업** 을 생성한다.

---

## ① Inception Keyword (사람 입력)

> 한 단어 또는 5 단어 이내. 명사보다는 *세계가 함의되는 구절* 이 좋다.
> 채운 뒤 `frontmatter.planted_at` 에 작성 시각을 적고, gen-001 시작 의지가 확정되면 `sealed: true` + `sealed_at` 으로 봉인.

```
<여기에 키워드 / 구절 한 줄>
```

## ② Brief — 사용자 의도 (planted 2026-04-29)

- **무엇을**: 콘텐츠 파생의 모태가 되는 가상 세계.
- **세계의 결**: 흥미 · 깊이 · 공명 — 사람들이 *"거기 살고 싶다 / 더 알고 싶다"* 가 되는 세계.
- **고객**: 상상하는 것을 좋아하고 즐기는 사람들.
- **벤치마크**: 해리포터 · 반지전쟁 · 아이언맨 · 슈퍼마리오 · 파이널판타지 · 주토피아.
- **작가 관심 (영양분)**: SF · AI · 과학 · 게임 · 판타지.
- **파생 형태 (예시)**: 게임 · 웹툰 · 영화 (트리트먼트) · 소설 · 이미지 · 우표.
  형식은 시스템이 추가 제안 가능 (Type B 결정).

## ④ 봉인 절차

1. 사용자가 ① 에 시드를 적는다.
2. `current.md` 에 `gen: 1`, `started_at: <iso>` 작성.
3. 본 파일 frontmatter `sealed: true`, `sealed_at: <iso>` 로 변경.
4. 봉인 후 system 이 `generations/gen-001/orgs/` 에 7±2 개 조직 제안을 Type C 결정으로 발의.

---

## ⑤ 봉인 후 시스템이 할 일 (참조)

- 본 시드 + `constitution.md` + `CHARTER.md` 를 읽고 gen-001 조직을 7±2 개 제안.
- 각 조직 charter 초안 (`generations/gen-001/orgs/<org>.md`) 작성.
- 사람이 Type C 결정으로 승인 → 첫 사이클 시작.
- 모든 활동은 `cycles/cy-NNN/ticks/` 에 기록.
