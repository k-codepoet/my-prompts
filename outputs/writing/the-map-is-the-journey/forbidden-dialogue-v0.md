# 여정 자체가 지도 — 금지 대사 사전 v0 (인물 대사 단위 그물)

> 본 사전은 *대사 단위* 의 자기 트립. forbidden-language-v0 (loremaster r2) 가 *세계의 입* (서술자 / 메타 본문) 의 단어를 그물로 거는 자리라면, 본 사전은 *인물의 입* 의 결을 그물로 거는 자리.
>
> **두 사전의 짝**: 같은 단어가 서술자에게는 *금지* 여도 인물 대사에서는 *그 인물답게* 들리기 위해 허용되어야 할 때가 있다 (예: 나림이 *항상 무서웠어* 라고 말할 때 — 본인의 환상 결을 박는 자리). 본 사전은 그 *허용의 결* 과 *그래도 안 되는 결* 을 인물 단위로 분리.
>
> 사용 시 *조직 단위 1 차 자기 검수*. writer · designer (게임 다이얼로그 공동 영역) 가 산출 직후 본 사전의 §3 (인물별) + §1 (매니페스토) + §5 (설교 4 종) 그물 통과. voice-keeper r2+ 측정 *전* 의 책임.
>
> charter §트립 #3 (*매니페스토 키워드가 대사로 직접 등장* → 설교가 됨) 의 정식 사전화. character-sheets-v0+axis-v0+character-relations-v0 의 8 자 인물 풀이 본 사전의 *인물 단위* 조각의 1 차 입력.

## §1. 매니페스토 7 키워드 *대사 직접 인용* 금지 (writer charter §트립 #3 정식)

bible v0.3 §해석 키 의 7 키워드 — `진심·움직임·자립·연대·축복·시간이 곧 나·방향` — 본 세계 인물 누구도 *이 단어를 그대로 입에 올리지 않는다*. 단어로 박는 순간 *결* 이 *슬로건* 으로 옅어지기 때문 — 본 세계의 모든 결은 *말로 박는 자리* 가 아니라 *살아낸 자리*.

| 키워드 | 인물 대사 안 사용 | 대체 표현 (본 세계 결로 박는 인물 대사) |
|--------|-----------------|----------------------------------------|
| **진심** | 금지 | "오래 들여다본 자리에서" / "그 새벽이 가벼웠던 적이 없어" / "한 번 본 사실은 가벼운 일이 아니야" |
| **움직임** | 금지 | "걸어간 자리만큼만" / "발끝의 방향이 같으면" / "하루 한 자국이라도" |
| **자립** | 금지 | "자기 손바닥은 자기만 펴" / "내 종이를 누가 본 적 없어" / "그 자리는 본인이 답할 자리고 내가 답할 자리는 아니야" |
| **연대** | 금지 | "옆에 앉을 사람이 있어야 한다" / "묻지 않은 채 같은 길을 걸었어" / "겹친 자리는 두 사람의 것" |
| **축복** | 금지 | "받은 새벽" / "백지를 받았다는 점에서만 같다" / "기쁘기도 무섭기도" (가벼운 결로 박을 때) |
| **시간이 곧 나** | 금지 | "매일 30 분이면 충분해" / "내가 무엇인지 보는 것은 5 분이면 된다" / "발끝에 쏟은 시간이 종이 위에 있다" |
| **방향** | 금지 (대사) — 단 *인물의 자기 인지* (예: *발끝의 방향*) 의 일상어 결로 등장 시 허용 | "발끝이 같은 곳을 가리킨다" / "어느 자리로 향해 있어" / "방향만 같으면 시간차는 결이 아니야" (이때 *방향* 은 일상어, 매니페스토 인용 아님) |

**§1 의 자기 트립**: 본 § 의 *키워드 자체* 가 본문 안에 등장 — 그러나 *명시 대상* 의 자리. 본 사전의 §1 사용은 *사전이 키워드를 키워드로 부르되 결로 박지 않음* — voice-keeper audit-rules §3 *배반 5 종 §1 직접 인용* 의 *예외 허용 = 메타 표시* 결과 동일 분류. 본 사전 본문이 그 외의 자리에서 7 키워드 직접 인용 = 0/7.

## §2. *큰 단어* 의 인물 대사 적용 룰 (loremaster forbidden-language §1~§8 의 인물 대사 변주)

forbidden-language-v0 의 §1~§8 키워드 (*영원·어디에나·절대적·깔린 길·모두 똑같이·영웅·항상·완전히*) 는 *서술자에게는 금지*. 인물 대사 안에서는 다음 룰로 한정:

| forbidden § | 키워드 | 인물 대사 안 사용 | 인물 단위 적용 룰 |
|-------------|------|-----------------|-----------------|
| §1 | 영원·영원히·끝나지 않는 | **조건부 허용** | 인물의 *환상·미숙* 으로만. *14 자국째 해 미만* 의 인물 또는 *한 번 본 사실의 충격* 안에 있는 인물 (나림 8 자국째 직후 결) 이 *그 새벽 이전의 자기로는 영원히 돌아갈 수 없어* 같은 자리에서. *해온 27 자국째 / 인규 42 자국째 / 연강 60+ 자국째* 는 *영원* 입에 올리지 않음. |
| §2 | 어디에나·모든 곳·전부 | **금지** (모든 인물) | *부정문* (*어디에도 없는*) 만 가능. 본 세계의 *없는 일이 된다* 결과 짝. |
| §3 | 절대적·완벽한 지도·완성된 지도 | **금지** (모든 인물) | *지도* 라는 단어는 인물도 *각자의 종이* 결로만. *절대적 / 완벽 / 완성* 은 어떤 인물도 입에 올리지 않음. *첫 거짓말* 은 인물도 안 함. |
| §4 | 깔린 길·준비된 길 | **금지** (모든 인물) | 인물도 *길* 단어 자체가 거의 없음. 정해 / 희재 의 *발끝의 방향* 결로 대체. *길* 이 등장하는 유일한 자리 = 인물이 *본 세계의 결을 오해한* 신호 (자기 인식 박음). |
| §5 | 모두 똑같이·동등하게 주어진·공평한 출발 | **조건부 허용** | *받은 새벽은 모두에게 같다* 한정구 안에서만. 그 외의 자리에서 *모두 똑같이* 입에 올리는 인물은 본 세계의 *각자의 종이가 다 다르다* 결을 오해함 — 자기 트립 (인물 자기 박음). |
| §6 | 영웅·주인공·위인·대단한 사람 | **조건부 허용** | 인물이 *영웅* 단어를 입에 올릴 때 — 그 인물이 본 세계의 *지도-스승* 결을 *오해한* 신호. 본 8 인 풀 중 누구도 *영웅* 을 입에 올릴 자리 없음 (모두 본 세계의 결을 따른 자들). 외부 인물 (cy-002 도착 후보) 의 자기 트립 자리. |
| §7 | 항상·언제나·변함없이 | **조건부 허용** | 인물의 *환상 / 기억의 왜곡* 으로만. 나림 *항상 무서웠어* (8 자국째 한 번 본 새벽의 결) 가 본 8 인 풀 중 유일 자리. 그 외 인물의 *항상* 입에 올림 = 인물 자기 트립. |
| §8 | 완전히·끝까지·마지막까지 | **조건부 허용** | 인물이 *작은 끝* (한 새벽의 끝, 한 자국의 끝) 결로 박을 때만. 유리 (8 자국째 아이) 의 *오늘 새벽은 다 끝났어* 같은 자리. 어른 인물이 *마지막까지* 입에 올림 = 본 세계의 *답은 합의되지 않는다* 결과 충돌 — 인물 자기 트립. |

## §3. 8 인 인물별 *말하지 않는 패턴* (voice signature 금지 표)

각 인물의 voice signature 는 character-sheets-v0 + character-sheets-axis-v0 + character-relations-v0 §2 직접 흡수. 본 §3 은 *그 인물답게 들리는 결* 을 *말하지 않는 결* 의 거울로 박음.

### 3.1 해온 (海溫) — *나* 축, 매일 30 분, 27 자국째

**말하지 않는 패턴**:
- *단정 어미* 안 씀 — *"확실히 ~야"* / *"분명히 ~다"* / *"~인 것 같다"* 의 단정형. *"오늘은 그렇게 보였어"* 의 관찰형이 본인의 결.
- *결론 박는 단어* 안 씀 — *"결국엔 ~"* / *"~할 수밖에 없어"*. 본인은 *매일 다르게 본다*, *결국* 의 결이 본인에게 없음.
- *남의 새벽 평가* 안 함 — *"그건 잘못된 결이야"* / *"네가 틀렸어"*. 본인은 *자기 손바닥조차 매일 다르게 본다* — 다른 사람의 결을 단정할 자리가 본인에게 없음.

**위반 예시 (본 사전이 거름)**:
- ✗ "확실히 매일 보면 변해" — 단정형 + 결론. 본인 voice 아님.
- ✓ "오늘은 어제와 미세하게 달라 보였어" — 관찰형. 본인 voice.

### 3.2 정해 (正解) — *나* 축, 한 번도 안 함, 27 자국째

**말하지 않는 패턴**:
- *짧은 단정* 안 씀 — *"그래"* / *"맞아"* / *"아니야"* 의 1~2 음절 결단형. 본인은 *길고 천천한 문장*, *한 마디 안에 두 박자*.
- *이분 박음* 안 함 — *"맞고 틀리고"* / *"옳고 그르고"* / *"좋고 나쁘고"* 의 이항 대립. 본인은 *"~할 수도 있고, ~할 수도 있고"* 의 두 가지 가능성 결.
- *물음* 안 함 — *"네 지도 봤어?"* / *"어디까지 갔어?"*. 본인은 *묻지 않은 채 같은 길을 걷는* 결. 단 *자기 발끝의 방향* 만 단정 (예외 1 호).

**위반 예시**:
- ✗ "맞아, 그게 자유야" — 짧은 단정 + 결론. 본인 voice 아님.
- ✓ "자유의 결인지 두려움의 결인지는, 안 본 본인이 답할 자리고 내가 답할 자리는 아니야" — 두 가능성 + 자리 분리. 본인 voice.

### 3.3 나림 (那林) — *나* 축, 한 번 (8 자국째) 보고 6 년 멈춤, 26 자국째

**말하지 않는 패턴**:
- *위로의 거짓* 안 함 — *"괜찮아"* / *"다 잘 될 거야"* / *"걱정 마"*. 본인은 *작은 자기 지도가 무서운 사실* 안에 있음 — 거짓 위로의 결이 본인에게 없음. 단 *남의 자국 옆에서 한 번* 본인이 *괜찮아* 를 들었을 때 *그 한 마디가 자기 12 색조 박물관의 한 점이 됨* — 본인의 입으로는 안 함.
- *결단형 미래* 안 박음 — *"내일은 ~할 거야"* / *"앞으로는 ~"*. 본인은 *6 년 동안 안 펴 본 새벽* 안에 있음 — *내일* 의 결단을 본인 입에 올리지 않음. 본인의 미래는 *후보 새벽* (관계 그물 §3.2) 으로만 가리켜짐.
- *남에 대한 단정* 안 함 — *"그 사람은 ~한 사람이야"*. 본인은 *12 색조 박물관* 의 사람들을 *직접 본 적 없음* — 자국으로만 안다 — 인물 단정의 자리가 본인에게 없음.

**위반 예시**:
- ✗ "괜찮아, 다음 새벽엔 잘 될 거야" — 위로 거짓 + 결단형 미래. 본인 voice 아님.
- ✓ "한 번 본 새벽 뒤의 새벽은 첫 번째와 다른 결인지, 그저 두 번째인지 — 둘 다인 것 같기도 해" — 망설임 + 두 가능성. 본인 voice.

### 3.4 유경 (柳敬) — *가족* 축, 35 자국째 다시, 38 자국째

**말하지 않는 패턴**:
- *희생어* 안 씀 — *"엄마니까"* / *"내가 다 해줄게"* / *"너만 잘 되면 돼"*. 본인은 *합쳐짐이 아닌 겹쳐짐* 의 결 — *자기 손바닥을 다시 펴는 자리* 를 박은 본인. 자기 자국 위에 *자기* 를 안 박는 희생어가 본인에게 없음.
- *아이의 결정 대신 박음* 안 함 — *"넌 ~해야 돼"* / *"그건 안 돼"*. 본인은 *"네가 좋다면 / 내가 좋다면"* 의 분리형 — 자기 의견을 자기에게만 단정.
- *과거 후회의 결단* 안 박음 — *"그때 안 펴본 걸 후회해"*. 본인은 *25–35 자국째 10 년 안 본 자리* 를 *없는 일이 아닌 다시의 자리* 로 박음 — 후회의 결단형이 본인 voice 아님.

**위반 예시**:
- ✗ "엄마니까 내가 너 위해서 다 안 봤어" — 희생어 + 단정. 본인 voice 아님.
- ✓ "내 종이 위에 한 아이가 그린 자국이 없었다면, 나는 내가 누구인지 더 몰랐을 거야" — 결의 박음 (희생 아닌 받음). 본인 voice.

### 3.5 인규 (仁圭) — *업* 축, 28 자국째 부터, 42 자국째

**말하지 않는 패턴**:
- *가르침어* 안 씀 — *"내가 가르쳐줄게"* / *"답은 ~야"* / *"이건 이렇게 하는 거야"*. 본인은 *지도-스승* 자리를 *받아들이지 않음* — *답을 박지 않음* 이 본인의 결 (bible §6.1 *통과의 산형 비극* 격하).
- *경험 박음* 안 함 — *"내가 28 자국째부터 옆에 앉아 봤는데 ~"* 의 권위형. 본인은 *옆에 앉을 사람의 새벽보다 자기 새벽이 짧아야* 의 결 — 자기 자국 분량을 입에 올리지 않음.
- *옳다/그르다 단언* 안 함 — 단 *"그건 아닌 것 같아"* 의 짧은 한 마디만 본인의 유일 단언 자리. 그 외 *옳다/그르다* 의 결단형은 본인의 voice 아님.

**위반 예시**:
- ✗ "내가 답을 알려줄게 — 매일 보는 게 답이야" — 가르침어 + 결론 박음. 본인 voice 아님.
- ✓ "네가 본 것이 무엇인지" — 묻기. 본인의 1 차 voice.

### 3.6 연강 (連岡) — *업* 축, 60+ 자국째, *짧음의 결* 1 분

**말하지 않는 패턴**:
- *길게 말함* 안 함 — 한 발화 ≤ 2 문장. *짧음의 결* 이 본인의 1 차 voice signature. 인규의 voice (한 마디가 짧아서 무겁다) 의 *60 자국째 변주* — 더 짧음.
- *답 박음* 안 함 — 본인은 *지도-스승* 자리를 *받아들이지도 거부하지도 않음*. 어떤 질문에도 *답* 을 직접 박지 않음. *되묻기* (예: *"본인은 어떻게 본 것 같아?"*) 가 본인의 1 차 응답 결.
- *자기 자국 분량* 입에 안 올림 — 본인의 종이 위 *600 명 옆자리 자국* 의 사실, 본인의 *60+ 자국째 해* 의 사실, 본인은 *그것을 모른다*. 본인의 자기 인지가 *어디까지가 자기 자국인지 모호한 자리* 자체.

**위반 예시**:
- ✗ "60 년 옆에 앉아 봐서 아는데 — 그건 이렇게 해" — 길게 + 권위 + 답. 본인 voice 의 모든 결과 정반대.
- ✓ "본인은 어떻게 본 것 같아?" — 1 문장 + 되묻기. 본인 voice.

### 3.7 희재 (熙在) — *나* 축, 한 번도 안 함 (정해와 동일 결), 27 자국째

**말하지 않는 패턴**:
- *확신어* 안 씀 — *"확실히"* / *"분명히"* / *"틀림없이"*. 정해와 같은 결 — 단 정해보다 *짧음* 도 더 미세함 (*시도조차 안 한* 결).
- *남에 대한 평가* 안 함 — *"그가 ~한 사람이야"*. 정해와 함께 두 번 같은 길을 걸으면서 *내 지도 봤어?* 안 물은 결 — 다른 사람에 대한 단정의 자리가 본인에게 없음.
- *결단형 약속* 안 박음 — *"세 번째도 같이 걸을게"*. 본인은 *세 번째는 둘 중 한 사람이 자기 발끝의 방향을 바꾸는 새벽에 일어나겠지* 의 *나란함* 결 — 박지 않은 채 같은 자리에 있음.

**위반 예시**:
- ✗ "확실히 우리는 같은 길을 또 걸을 거야" — 확신 + 결단형 약속. 본인 voice 아님.
- ✓ "두 번 걸었으면 충분해. 세 번째는 ~ 새벽에 일어나겠지" — 비결단형 + 가능성. 본인 voice.

### 3.8 유리 (柳里) — *가족 → 나* 축, 7 자국째 부터, 8 자국째 (아이)

**말하지 않는 패턴**:
- *고정어* 안 씀 — *"절대 안 ~"* / *"다신 ~"* / *"늘 ~"*. 본인은 *자라남의 결* — 본인의 결이 *고정* 되는 단어가 본인에게 없음. 단 *어머니의 결을 흉내낼 때* 의 자리에서만 *어머니가 그렇게 말하니까* 의 일시적 흉내 (인물 자기 트립 1 호).
- *추상어* 안 씀 — *"의미"* / *"진실"* / *"본질"*. 8 자국째 아이의 voice 는 *구체* 만. *손바닥 / 새벽 / 엄마 / 흙 / 색조* 의 일상어로만.
- *남의 평가 단정* 안 함 — *"엄마는 ~한 사람이야"*. 본인은 *엄마가 안 슬퍼했어* 같은 *관찰* 만 — 단정의 자리가 본인의 결에 없음 (어머니 voice signature 직접 흡수).

**위반 예시**:
- ✗ "엄마는 절대 안 슬퍼해 — 늘 강한 사람이야" — 고정어 + 단정. 본인 voice 아님.
- ✓ "엄마는 내가 엄마 손을 안 잡고 자기 손금 새벽을 시작한 새벽에 안 슬퍼했어" — 구체 관찰. 본인 voice.

## §4. *세 축 jargon* 금지 (axis 결을 말로 박지 않음)

bible v0.3 §해석 키 의 *나 / 가족 / 업* 세 축은 *분류* 가 아니라 *결*. 인물이 자기 축을 *말로 박는 자리* — *"나는 가족 축 사람이야"* / *"이건 업 축 결정이야"* — 는 본 세계 인물 누구도 안 함. 축은 *살아낸 자리* 로만 드러남.

| 축 | 인물 대사 안 *jargon* | 대체 (살아낸 자리로 박는 인물 대사) |
|-----|---------------------|----------------------------------|
| 나 | *"나는 나를 위해 ~"* / *"이건 내 일이야"* | "내 손바닥은 매일 30 분이면 충분해" / "내 발끝의 방향만은 매일 같아" |
| 가족 | *"가족이니까 ~"* / *"엄마는 ~"* (희생어) / *"부모로서 ~"* | "내 종이 위에 한 아이가 그린 자국이 없었다면" / "엄마 손을 안 잡고 자기 손금 새벽을 시작한 새벽" |
| 업 | *"이건 내 일이야"* / *"전문가로서 ~"* / *"내 업이 ~"* | "옆에 앉을 사람의 새벽보다 자기 새벽이 짧아야" / "본인이 그렇게 부르면 그렇게 듣는다" |

**§4 의 자기 트립**: 본 사전 본문 §3·§4 안의 *나·가족·업* 단어 사용은 *분류 도구* 의 자리 — 인물 voice 박음 자리 아님. 본 사전이 *축* 을 *축 분류* 자리에서 사용하는 것은 voice-keeper audit-rules §1-e *세 축 부재 임계* 의 *측정 입력* 결과 동일 분류.

## §5. *설교 4 종* 의 인물 대사 변주 금지 (voice-keeper audit-rules §4 짝)

voice-keeper audit-rules-v0 §4 *설교 4 종* (1) 명령형 조언 (2) 의미 박는 결론 (3) 단정형 결단 (4) 도덕화 — 는 *서술자* 측면. 인물 대사로 *설교* 가 들어오는 자리도 본 사전이 거름. 본 8 인 풀 중 누구도 다음 4 종 결로 발화 안 함:

| 설교 종 | 인물 대사 안 패턴 | 본 8 인 voice 안 응답 결 |
|--------|-----------------|------------------------|
| **§5.1 명령형 조언** | *"~해야 돼"* / *"~하지 마"* / *"~하는 게 좋아"* | 묻기 (인규: *"네가 본 것이 무엇인지"*) / 두 가능성 박음 (정해/희재/나림) / 자리 분리 (해온: *"본인이 답할 자리고 내가 답할 자리는 아니야"*) |
| **§5.2 의미 박는 결론** | *"그래서 ~인 거야"* / *"결국 ~라는 의미야"* / *"이게 ~의 결이야"* | 관찰 (해온: *"오늘은 그렇게 보였어"*) / 자리 박음 (유리: *"엄마는 ~ 새벽에 안 슬퍼했어"*) — 의미 박지 않은 채 자리만 |
| **§5.3 단정형 결단** | *"확실히 ~"* / *"분명히 ~"* / *"틀림없이 ~"* | 두 가능성 (정해/희재) / 망설임 (나림) / 부드러운 단정 (인규: *"그건 아닌 것 같아"*) |
| **§5.4 도덕화** | *"그건 옳아"* / *"그건 잘못이야"* / *"~해야 옳아"* | 자리 분리 (인규: *"옳다/그르다 단언 안 함"*) / 결의 박음만 (유경: *"기쁘기도 무섭기도"*) / 결을 결로 두기 (연강: *"본인은 어떻게 본 것 같아?"* 되묻기) |

본 8 인 풀이 본 §5 4 종에 *전 인물 무응답* — 본 세계의 voice 가 *설교 결* 자체와 1 차 격하 짝. *설교가 들어올 자리* 가 본 8 인 안에 없음 = bible v0.3 §6.1 *통과의 산형 비극 격하* 의 *대사 단위 1 차 안전핀*.

## §6. 사용법

### 6.1 1 차 자기 검수 (조직 단위)

산출물 (단편 / 게임 다이얼로그 / 인물 시트 신규) 작성 직후, 본 사전의 §1·§3·§5 grep:

```
# §1 매니페스토 7 키워드 (인물 대사 안)
진심|움직임|자립|연대|축복|시간이 곧 나|방향

# §5 설교 4 종 패턴 (인물 대사 안)
~해야 돼|~하지 마|~하는 게 좋아|결국|확실히|분명히|틀림없이|옳아|잘못이야

# §3 인물별 — 각 인물 자기 voice 의 *말하지 않는 패턴* 표 직접 확인
```

적중 ≥ 1 → §1·§3·§5 의 *대체 / 인물 단위 적용 룰 / 위반 예시* 검사. 통과 표 안의 자리는 통과, 그 외 자리는 대체.

### 6.2 2 차 통과 기준 (voice-keeper r2+ 측정 *전*)

- 본 사전 미통과 산출물은 voice-keeper *배반 5 종 §1 직접 인용* / *설교 4 종* 임계로 자동 흘러감.
- 본 사전은 *대사 단위* 그물. *맥락 단위 위반* (예: 인물이 본 사전 통과한 단어로 *결의 의미를 박는 자리*) 은 voice-keeper r2+ 가 봄.
- forbidden-language-v0 (서술자) + 본 사전 (인물 대사) = 2 그물 동시 통과 의무.

### 6.3 사전 갱신

- 새 위반 자리 발견 시 writer 에 보고 → v0 → v0.1 갱신.
- 신규 인물 (cy-002 *2 차 인물 풀* 도착 시 — 인규의 옆 200 인 / 유리의 옆 친구들 / 정해의 다른 동행자) 도착 시 §3 인물 단위 표 행 추가.
- character-sheets / character-relations 갱신 시 본 사전 §3 표 *재흡수* 의무.
