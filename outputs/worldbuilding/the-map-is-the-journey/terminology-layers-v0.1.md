## §7. plain-Korean 치환표 (toxic 15 자리 → 일상어 변주)

> **자리.** reader-first-standard.md §3 toxic-term budget = 15 자리. writer 가 한 단락 안 ≥ 2 자리 박을 경우 = §3 hard-rule 발화. 본 §7 = *어휘를 버리지 말고 층위를 골라라* 도구. 같은 의미를 *일상어* 로 박을 자리 (= reader-first §2 *plain confession* 정합) 와 *세계 단어* 로 박을 자리 (= 본 세계 작동 단어, *손금 새벽* / *자국빛* 등 핵심 자리) 를 분리.

> **사용 룰.** ① 첫 500 자 = plain 측 단독 박음 의무 (toxic 0). ② 한 단락 안 ≥ 2 자리 박음 = plain 측 1 회 이상 치환. ③ *손금 새벽* / *자국빛* 두 핵심 자리는 *층위 표지 1 행* 박음 후 1 회 박음 가능 (v0 §3 처치). ④ 나머지 13 자리는 plain 변주가 *기본*, 세계 단어가 *변주*.

## §8. toxic 15 자리 풀 5 층위 매트릭스 (v0 §1 + §4 후보 흡수)

v0 §1 = 7 어휘 5 층위. 본 v0.1 = +8 어휘 흡수 = 15 어휘 5 층위 풀 (reader-first §3 1:1).

| 층위 | 어휘 | 비고 |
|------|-----|------|
| **시간** (time) | 손금 새벽 (◎) / 자국 (∗ 시간 단위) / 새벽 (◎) | 새벽 = *손금 새벽* 의 *새벽* 만 단독 박음 자리. *후회의 새벽* / *자기 새벽* 등 합성어 자리 |
| **시각** (visual) | 자국빛 (◎) / 색조 (◎) | 색조 = *흙 색조* 자리 |
| **사물 / 단위** (thing / unit) | 손금 (◎) / 자국 (∗ 사물) / 흙 (◎) / 종이 (◎) / 손바닥 (◎) / 지도 (◎) | 종이 = *받은 백지* / *자기 종이*. 손바닥 = *손금* 의 사물 자리. 지도 = *자기 지도* 합성어 자리 |
| **행위 / 상태** (act / state) | 박음 (◎ 행위) / 박힘 (◎ 상태) | 박음 = *자국을 박는다* 단독. 박힘 = *자국이 박혀 있다* 단독 |
| **추상 / 속성** (quality) | 결 (◎) / 자리 (∗ 다층 — 사물 + 속성) / 박자 (∗ 다층 — 속성 + 시간) / 본인 (◎ 인칭) / 옅게 (◎ 부사) | 자리 + 박자 = 다층 박음 (자국 같은 ∗ 표지). 다른 6 자리는 단층 박음 의무 |

**∗ 표지 자리 4 자리** (다층 박음): 자국 (시간 + 사물) / 자리 (사물 + 속성) / 박자 (속성 + 시간). 다른 11 자리 = 단층 박음 의무 (한 어휘 한 층위).
