# 금지 표현 사전 v0 — *본 세계가 말하지 않는 것*

> 본 사전은 *언어 단위의 자기 트립*. 본 세계는 *정지·전능·완결* 결의 단어를 본문 안에서 쓰지 않는다 — 그것이 본 세계의 자연법칙 (걸어야 자란다, 자기만 자기 손바닥을 편다, 답은 합의되지 않는다) 을 *말 한 마디로* 무너뜨리기 때문.
>
> 사용 시 *조직 단위 1 차 자기 검수*. designer · writer · art-director · implementer 가 산출물 끝낸 직후 본 사전 grep. 적중 시 *대체* 또는 *맥락 한정* (출처 명시 + 인물 대사로만).
>
> voice-keeper 정렬 감사 룰 v0 §3 *배반 5 종* 과 결 일치. 본 사전은 *배반 적중 전* 의 *언어 단위* 1 차 그물.

---

## §1. *영원* / *영원히* / *끝나지 않는*

- **왜 금지**: 본 세계의 자국은 *느리게라도 옅어진다* (chronicle v0 사건 5 *해 없음* 참고). *영원* 은 자국이 *지워지지 않는다* 는 거짓을 박는 단어 — *움직임 = 자라남* 의 자연법칙 자기 모순.
- **대체 후보**:
  - "오랫동안" / "긴 동안"
  - "지워지지 않은 채로 (그 자리의 사람이 그 자리에 있는 한)"
  - "비에도 굳건한"
- **예외 허용**: 인물 대사 안에서 *그 인물의 환상 / 미숙* 으로 쓰일 때만. 작가 코퍼스 직접 인용 시 출처 명시.

## §2. *어디에나 있는* / *전부 / 모든 곳*

- **왜 금지**: 본 세계의 자국은 *그 사람이 걸은 자리에만* 있다. *어디에나 있는* 은 *움직임 없이 도달함* 의 거짓 — 본 세계의 *방향* 결을 무너뜨림.
- **대체 후보**:
  - "여러 자리에"
  - "걸어간 자리들에"
  - "겹친 자국이 박힌 자리들에"
- **예외 허용**: 부정문 (*어디에도 없는*) 은 사용 가능 — 본 세계의 *없는 일이 된다* 와 짝.

## §3. *절대적인 지도* / *완벽한 지도* / *완성된 지도*

- **왜 금지**: 본 세계에는 *지도* 라는 단어가 *각자의 종이* 에만 쓰인다. *절대 / 완벽 / 완성* 은 *모두에게 같은 지도가 있다* 는 거짓 — 본 세계의 *자립* 결과 정면 충돌.
- **대체 후보**:
  - "자기 종이의 한 단면"
  - "오래 걸은 자국"
  - "지도-스승의 종이"
- **예외 허용**: 없음. *절대적인 지도* 는 본 세계의 *첫 거짓말*.

## §4. *어디에나 있는 길* / *깔린 길* / *준비된 길*

- **왜 금지**: 본 세계는 *길* 이라는 단어를 거의 쓰지 않는다 (terrain-v0 *통과의 산* 참고). *길* 은 *사람이 만드는 것* 이지 *세계에 깔린 것* 이 아님. *깔린 / 준비된* 은 *움직이기 전에 이미 있는 것* 의 거짓.
- **대체 후보**:
  - "여럿이 지나간 자리"
  - "통과의 산 가장자리"
  - "겹친 자국이 모인 곳"
- **예외 허용**: 인물이 *길* 이라는 단어를 쓸 때 — 그러나 작가 시점 (서술자) 은 사용 금지.

## §5. *모두 똑같이* / *동등하게 주어진* / *공평한 출발*

- **왜 금지**: 본 세계의 *백지의 새벽 = 평등* (bible §2.1) 은 *받았다는 사실의 평등* 이지 *내용의 평등* 이 아님. *모두 똑같이* 는 *각자의 종이가 다 다르다* 는 본 세계의 *자립* 결과 충돌.
- **대체 후보**:
  - "받은 새벽은 모두에게 같다"
  - "백지를 받았다는 점에서만 같다"
  - "각자의 떨림은 각자의 것"
- **예외 허용**: *백지를 받았다는 사실 자체* 라는 한정구와 함께 쓸 때만.

## §6. *영웅* / *주인공* / *위인* / *대단한 사람*

- **왜 금지**: 본 세계에는 *지도-스승* 이 있을 뿐 *영웅* 이 없다. *영웅* 은 *남의 종이가 자기 종이보다 큰 의미가 있다* 는 거짓 — 본 세계의 *자기 손바닥은 자기만 편다* 결과 충돌. visual-bible v0.1 의 *영웅 포즈 금지* 와 짝.
- **대체 후보**:
  - "오래 걸어 굳은 자리의 사람"
  - "지도-스승"
  - "사랑의 산이 큰 사람"
- **예외 허용**: 인물 대사로 *영웅* 이라는 단어가 쓰일 때 — 그 인물이 본 세계의 결을 *오해한* 신호로만.

## §7. *항상* / *언제나* / *변함없이*

- **왜 금지**: 본 세계의 *자국* 은 *진하기* 와 *면적* 으로 변한다. *항상* 은 *변하지 않음* 의 거짓 — *시간이 곧 나* 결과 충돌.
- **대체 후보**:
  - "오래"
  - "여러 새벽 동안"
  - "그 자리에 있는 한"
- **예외 허용**: 인물의 *환상* 또는 *기억의 왜곡* 으로 쓰일 때만.

## §8. *완전히 / 끝까지 / 마지막까지*

- **왜 금지**: 본 세계는 *끝* 이 약하다 — chronicle v0 사건 5 *해 없음* 참고. *완전 / 끝 / 마지막* 은 *완결* 의 거짓 — *답은 합의되지 않는다* 결과 충돌.
- **대체 후보**:
  - "어느 자국까지"
  - "그 새벽까지"
  - "옅어지기 전까지"
- **예외 허용**: 한 새벽의 끝 (해가 들면 손바닥의 그림이 사라짐) 같은 *작은 끝* 에 한해 사용.

## §사용법

### 1) 1 차 자기 검수 (조직 단위)

산출물 (게임 텍스트 / 단편 / 시각 캡션 / charter 본문) 을 끝낸 직후, 본 사전의 §1~§8 키워드로 본문 grep:

```
영원|영원히|끝나지 않는|어디에나|모든 곳|전부|절대적|완벽한 지도|완성된 지도|깔린 길|준비된 길|영웅|주인공|항상|언제나|변함없이|완전히|마지막까지
```

적중 ≥ 1 건 → §1~§8 의 *대체 후보* 또는 *예외 허용* 검사. 예외 허용 외의 자리는 대체.

### 2) 2 차 통과 기준

- voice-keeper r2 측정 *전* 에 본 사전 통과가 *조직 단위 책임*. 본 사전 미통과 → voice-keeper 자동 *배반 5 종 / 설교 4 종* 임계로 흘러감.
- 본 사전은 *언어 단위 그물*. 그물에 안 걸리는 *문맥 단위 위반* (예: *멈춤이 곧 평화* 의 명제 자체) 은 voice-keeper 가 봄.

### 3) 사전 갱신

- 새 위반 자리 발견 시 loremaster 에 보고 → 본 v0 → v0.1 갱신.
- 작가 코퍼스 (`~/k-codepoet/my-essay/content/`) 에서 본 사전과 *충돌하는 표현* 이 발견되면 *작가 코퍼스 우선* — 본 사전이 갱신됨 (작가 코퍼스가 본 세계의 1 차 진실).

## 작가/시스템 메모 (cold reader 는 본 § 안 읽어도 됨)

- 본 v0 는 *8 종* 의 1 차 골격. v0.1 에서 *부드러운 트립 후보* (예: *영혼* / *운명* / *천국* — 본 세계의 *자국·종이·새벽* 에 들어맞지 않는 어휘) 추가 검토.
- 본 사전이 *너무 빡빡* 해질 위험: 본 세계의 *축복* 결을 표현할 때 *떨림 / 선물 / 따뜻함* 같은 단어는 본 사전과 충돌하지 않음 (bible v0.2 §2.1 직접 참조). voice-keeper r2 측정 시 *축복* 결이 본 사전 때문에 옅어졌는지 점검 필요 (Trap 3 자기 검사 — bible §9.2).
- 본 사전과 charter §트립와이어 의 *영구 보존된 도서관* 예시는 같은 결 — 본 사전이 *어휘 단위*, 트립와이어가 *사물 단위*. 두 그물이 짝.
- designer / writer / art-director 가 본 사전을 *손쉽게* 쓰려면 IDE/에디터 검색 매크로 형태로 가공 필요. 그러나 그것은 implementer 영역 — 본 라운드 인계만.
