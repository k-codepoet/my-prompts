## 1. 7 어휘 × 5 층위 분리표

| # | 어휘 | 층위 | 한 줄 정의 | 사용 예 (정합) | 사용 금지 (혼동) |
|---|------|------|----------|------------|--------------|
| 1 | **손금 새벽** | 시간 (time) | *매일 30 분, 자기 손바닥을 펴 자기 자국을 보는 시간* (bible §4) | "그는 14 자국째부터 손금 새벽을 빠뜨리지 않았다" | "손금 새벽이 흐른다" (시간 → 사물 혼동) / "손금 새벽이 비친다" (시간 → 시각 혼동) |
| 2 | **자국빛** | 시각 (visual) | *자기가 방금 박은 자국 1~3 점이 0.5 s 동안 옅게 빛남* (regions v0 §5) | "발끝에서 자국빛이 잠깐 비쳤다" | "자국빛 시간을 놓쳤다" (시각 → 시간 혼동) / "자국빛으로 갔다" (시각 → 행위 혼동) |
| 3 | **손금** | 사물 / 은유 (thing-as-metaphor) | *사람의 종이에 박힌 자국 패턴 전체* — 손바닥 안 손금이 *그 사람의 지도 자체* | "그의 손금은 깊고 좁았다" | "손금을 했다" (사물 → 행위 혼동) — *손금 새벽을 했다* 가 정합 |
| 4 | **자국** | 사물 / 단위 (unit) | *한 걸음에 박힌 흙 한 점* — 본 세계의 *시간 단위* (N 자국째) 와 *사물 단위* 동시 | "5 자국째 / 200 자국째" (시간) / "그의 자국은 옅었다" (사물) | "자국이 비친다" (사물 → 시각 혼동, 자국빛 의 자리). 자국 자체는 빛나지 않는다 — *자국빛* 만 빛난다 |
| 5 | **박음 / 박힘** | 행위 / 상태 (act / state) | *자국을 박는 동사* (박음) + *자국이 박혀 있는 상태* (박힘) | "그는 한 자국을 박았다" / "그 자국은 종이에 박혀 있다" | "박음이 비친다" (행위 → 시각 혼동) / "박음이 새벽에 일어난다" (행위는 *시간 위에* 일어나지 시간 *자체* 가 아님) |
| 6 | **결** | 추상 / 속성 (quality) | *종이 / 사람 / 자리의 성질* — 흐름의 방향성 + 질감 동시. 5 지역 *결* (음습한 길 결 / 의지소 결 등) 의 *결*. | "이 자리의 결은 음습한 길 결이다" / "그의 결은 짧다" | *결* 을 *자국* / *흙* 과 같은 사물로 사용 → reject (결은 *속성* 이지 *사물* 아님). "결을 박는다" → reject (속성은 박는 자리 아님 — *자국* 만 박는다) |
| 7 | **흙** | 사물 / 재료 (material) | *자국의 외형 재료* — 진하기 (= 진심) + 면적 (= 방향) + 색조 (= 사연) 의 단위 | "그의 흙은 진했다" / "두 흙이 겹쳤다" | "흙이 박힌다" → 부정확 — *자국* 이 박히고 *흙* 은 *자국의 외형*. *흙으로 박는다* → reject (흙 = 결과물의 재료이지 행위 도구 아님) |

## 2. 5 층위 자가 점검 (산출물 박음 시)

산출물 안 7 어휘 사용 시 다음 5 층위 중 *어느 자리* 인가 자가 답:

| 층위 | 표지 | 7 어휘 분포 |
|------|-----|-----------|
| 시간 (time) | *언제 / 몇 자국째 / 매일 30 분* | 손금 새벽 (◎ 단독) / 자국 (단위 변주 ○ — *N 자국째*) |
| 시각 (visual) | *어떻게 보이는가 / 비치는가* | 자국빛 (◎ 단독) |
| 사물 (thing) | *무엇이 어디에 있는가* | 손금 / 자국 (사물 변주) / 흙 |
| 행위 / 상태 (act) | *무엇을 하는가 / 무엇이 되어 있는가* | 박음 (◎ 단독) / 박힘 (변주) |
| 추상 / 속성 (quality) | *어떤 결인가 / 어떤 성질인가* | 결 (◎ 단독) |

> 7 어휘 모두 *1 층위 1 자리* 박음 (◎). 단 *자국* 은 시간 단위 + 사물 단위 *2 층위 박음* (∗ 표지) — 본 어휘만 다층 박음 자리. 다른 6 어휘는 단층 박음 의무.

## 3. 사용자 진단 사례 1 호 (F-1310 본문 직접 흡수)

R0 사용자 진단 (F-1310): "*'자국 새벽' 이라는 단어는 코퍼스 grep 0 건. 사용자는 '손금 새벽' (193 회) 과 '자국빛' (345 회) 를 한 추상 단어로 뭉쳐 들었다.*"

본 사례 = 시간 (손금 새벽) + 시각 (자국빛) 층위 *혼동* 의 1 호. 두 어휘 = 음운 유사 (*새벽* = 손금 새벽 / *빛* = 자국빛) + 의미 인접 (*자기를 봄*) + 산출물 안 위치 인접 → 사용자가 *자국 새벽* 으로 뭉쳐 들음.

처치:

- **산출물 박음 시 두 어휘 한 단락 안 인접 박음 금지** — 같은 단락에 손금 새벽 + 자국빛 동시 박음 = reject (혼동 트리거). 분리 박음 의무.
- **두 어휘 한 회차 안 둘 다 박음 시** = 첫 등장 자리에 *층위 표지* 1 행 박음 의무 (예: *손금 새벽 (시간 — 매일 30 분)* / *자국빛 (시각 — 잠깐 빛)*).

## 4. 다른 어휘 추가 후보 (cy-003+ v0.1 박음 자리)

본 v0 = 7 자리 1 차 박음. cy-003+ 후속 박음 후보 (사용자 진단 + 다른 조직 측정 시 추가):

| 후보 어휘 | 추정 층위 | 진단 |
|---------|---------|----|
| **자리** | 사물 / 공간 + 추상 / 속성 (다층) | "쉼의 자리" (사물 + 속성) / "그 자리" (사물) — 자국 같은 다층 박음 후보 |
| **박자** | 추상 / 속성 + 시간 (다층) | "박자가 빠르다" (속성) / "박자에 맞춰" (시간) — 다층 박음 후보 |
| **새벽** | 시간 (단독) | 손금 새벽 의 *새벽* 만 분리 박음 자리 — *해 없음의 새벽* / *후회의 새벽* 등 합성어 |
| **종이** | 사물 (단독) | *받은 백지* / *자기 종이* — 사물 단독 |
| **색조** | 시각 (단독) | *흙 색조* — 시각 단독, *자국빛* 과 분리 |
| **본인** | 추상 / 인칭 | *자기* 와 차이 — 본 v0.1 박음 시 분리 자리 |
| **옅게** | 추상 / 속성 부사 | *옅은 흙* (형용사) ↔ *옅게 옅어진다* (부사) — 분리 자리 |

> 본 §4 = v0.1 박음 자리 후보. cy-003 안 본 v0 = 7 자리 baseline. 추가 발견 어휘는 v0.1 박음 시점 흡수.

## 5. 트립와이어 (어휘 단위 발화 조건)

- 한 어휘를 *2 층위 이상 박음 (자국 / 박음·박힘 외 6 자리)* → reject. *층위 분리* baseline 위반.
- 한 단락 안 *손금 새벽 + 자국빛* 인접 박음 → reject (F-1310 사용자 진단 직접 처치).
- *자국 새벽* / *자국 빛* 과 같은 *합성 어휘 신규 발의* → reject (혼동 트리거 신규 박음 금지). 새 어휘 발의는 본 v0 표 위에 *분리 가능 검증* 선행.
- *결을 박는다* / *흙으로 박는다* → reject (행위 동사 *박음* 의 목적어 = *자국* 단독).
- *자국이 빛난다* → reject (자국 자체는 빛나지 않음 — *자국빛* 만 빛난다).
- 본 v0 외 *7 어휘 신규 정의 박음* → loremaster bible / 본 v0.1 개정 선행.
