# 어휘 매핑 v1 — 한국 일상 결로 톤 이동

> **본 v1 골자.** 사용자 R0 발의 7자리 누적 (F-20260502-1216 → F-20260503-* + D-20260503-001) 채택 후 1차 응답. 우화 결 (손금 새벽 / 자국 / 박힘 / 자국강 등 합성어) → 한국 일상 결 (신발 / 다녀온 자리 / 적다·남다 / 정선·김제·단양 등 실제 지명) 단일 진실 매핑.

> **읽는 자리.** loremaster·writer·critic·voice-keeper·art-director 가 신규 산출물 적기 전에 본 표를 먼저 읽는다. 단편 본문은 첫 1500자 안 메타포 단어 ≤ 3회 baseline (voice-keeper §7 + critic cold-read).

---

## 1. 의식 이름 — 폐기, 행동만

| 이전 (v0) | 이후 (v1) |
|----------|----------|
| 손금 새벽 (이름) | (이름 없음 — *매일 새벽 현관에서 신발을 한 번 본다* 행동만) |
| 손바닥 (의식 매개) | 신발 (현관 신발 한 켤레) |
| 손바닥을 펴다 | 신발 끝을 본다 |
| 손금 위 자국 | 신발 끝의 흙 (어제 다녀온 길의 색조) |
| 자기 종이를 본다 | 어제 다녀온 자리를 안다 |

**행동 묘사 baseline (이름 없음):**
> 매일 새벽 현관에서 신발 한 켤레를 한 번 본다. 어제 어디 다녀왔는지, 신발 끝에 묻은 흙으로 한 번 안다. 짧으면 30초, 길어야 2~3분.

이름이 없다 = 메타포로 굳지 않는다. 인물마다 *이 행동* 의 길이/태도/생략 여부가 voice-signature.

## 2. *자국* 의 평이화

| 이전 (v0) | 이후 (v1) — 문맥에 따라 골라 씀 |
|----------|---------------------------|
| 자국이 박혔다 | 흔적이 남았다 / 자리가 남았다 / 다녀온 표시가 남았다 |
| 자기 자국 | 어제 다녀온 자리 / 자기 길 / 어제 한 일 |
| 옆 사람의 자국 | 옆 사람이 다녀온 자리 / 옆 사람의 어제 |
| 자국이 굳다 (산이 되다) | 길이 굳다 / 길이 한 줄로 남다 / 같은 자리를 오래 다니면 길이 된다 |
| 자국빛 | (이름 폐기) — 풍경 단서로 풀어 씀: *방금 다녀온 자리가 잠깐 보인다* |

**시각 단서 보존**: *흙 색조* 는 그대로 (신발 끝의 흙으로 어제 어디 갔는지 보는 결).

---

## 3. *박다 / 박음 / 박힘* 분리

| 의미 | 이전 (v0) | 이후 (v1) |
|------|----------|----------|
| 시스템·기록 (산출물 만들기, 룰 정하기) | 박다 / 박음 | **적다** |
| 단편 안 흔적 (사람의 결이 어딘가에 남음) | 박히다 / 박힘 | **남다 / 남기다** |
| 결정·룰의 정착 | 정식 박음 / baseline 박음 | **자리잡다 / 자리잡히다** |

**예시:**
- v0: "writer r1 산출 박음" → v1: "writer r1 산출 적음"
- v0: "그 새벽이 자기 손바닥에 박혔다" → v1: "그 새벽이 그에게 남았다"
- v0: "정식 룰 baseline 박음" → v1: "정식 룰로 자리잡음"

## 4. *결 / 자리* 추상 합성 정리

| 회피할 합성 (v0) | 평이화 (v1) |
|-----------------|------------|
| ~의 결 | ~의 느낌 / ~의 방식 / ~ |
| 받음의 결 | 받았다는 느낌 / 받는 방식 |
| 가족의 결 | 가족 같은 느낌 / 가족 비슷한 어떤 것 |
| 짧음의 결 | 짧다는 점 / 짧은 방식 |
| 옆자리의 결 | 옆자리에 있는 일 / 옆에 앉는 일 |
| ~ 자리 (추상) | (보존 — *벤치 자리* / *식탁 자리* 등 구체는 그대로) |
| 받음의 자리 | 받는 사람 / 받는 쪽 |
| 옆자리 (구체 = 옆 의자) | 보존 |

**룰**: *결* / *자리* 는 *구체적 위치/물건* 일 때만 보존. 합성 추상 (*받음의 결*, *짧음의 자리*) 은 평이화.

## 5. 6 권역 — 한국 지명

| 이전 (v1 합성) | 이후 (v2 한국 지명) | terrain | 풍경 한 줄 | 거주 인물 |
|----------------|--------------------|---------|-----------|---------|
| 새벽재 | **정선** (강원) | 산골 — 비탈 한 줄 마을 | 비탈 위에 한 줄로 늘어선 마을, 깊은 새벽 안개, 골을 따라 한 방향 바람 | 정해 / 연강 / 희재 |
| 옅은들 | **김제** (전라) | 들녘 — 만경평야 | 평지에 분산된 마을, 사방 바람, 흙이 빨리 옅어짐 | 해온 (어릴 때) / 유리 (자라는 자리) |
| 느릿내 | **단양** (충북) | 강가 — 남한강 한 줄 마을 | 강 위 새벽 안개 한 방향 흐름, 윗집 부엌 빛 한 박자 늦게 아랫집 | 유경 / 유리 |
| 흙포구 / 마른포 | **통영** (경남) | 포구 — 작은 어촌 | 종이 가장자리, 떠남·돌아옴, 강한 한 방향 바람 | (현재 풀 0) |
| 굳은마을 / 늘재 | **무주** (전라) | 분지 — 산에 둘러싸인 정착 마을 | 산이 사방을 둘러쌈, 새벽이 늦게 도착, 정착 결 | 나림 |
| 빽빽골목 / 빽빽골 | **성남 변두리** | 도시 변두리 — 빽빽 골목 | 통과의 길이 가장 두꺼움, 자국 익명, 사방 발소리 | 인규 |

해온 도시 변두리 자리 = 김제 변두리 (들녘 ↔ 도시 경계) — 별도 매핑.

## 6. 메타포 룰 강화

**서술자 어휘 (단편 본문 1~3인칭 화자):**
- 평이한 한국어 baseline. 첫 1500자 안 메타포 단어 ≤ 3회.
- 금지 합성: 자국 박힘, 손금 새벽, 자국강, 받음의 결, 옆자리의 결, 짧음의 결 등.
- 허용: 흙 (구체 시각), 새벽 (시간), 길 (구체), 자리 (구체).

**인물의 행동·대사·생각 안 메타포:**
- 가끔만 (회차당 1~3회). 인물이 *지금 자기 결을 짧게 시적으로* 떠올리는 자리.
- 메타포가 인물 voice 의 일부일 때만 — 작가 voice 의 일부가 아님.

**측정:**
- voice-keeper §7 — 메타포 밀도 25% 이하 baseline 검수.
- critic cold-read — 첫 5분 안 *어디인지 / 누가 / 무슨 일* 즉답 + 메타포 단어 카운트.
- 사용자 R0 — 최종 PASS 게이트.
