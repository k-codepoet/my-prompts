### 2.4 동행 자국 hex (NEW — walk-trace-spec §M3 검수 결과)

동행 자국은 §2 먼지길 폭 *바깥* 의 *더 옅은 폭* 으로 격하 — `#9A8161 ~ #A89070`.

이유: 본 v0.2 §11 의 *나림* hex 패밀리 (`#7A6447 ~ #8B7355`) 와 동행 자국이 시각적으로 같아질 위험. 인물의 자국 ≠ 동행의 자국 이라는 결을 hex 단위에서 분리 보장. walk-trace-spec §M3 의 *동행 자국은 플레이어 자국보다 살짝 옅게* 룰을 *한 단계 더 옅게* 로 정밀화 (인계 §13).

### 2.5 인물 외형 hex 6 자리 예약 (v0.1 §7 → v0.2 §11 정밀화)

## 5 카메라 / 구도 규칙 (§5.4 NEW)

### 5.4 신설 컷 처방 3 종 — *손금 새벽 / 쉼의 자리 / 화살표 자국*

#### 5.4.1 *손금 새벽* 컷 (bible v0.2 §4)

본 컷의 1 차 시각 정의:

- **카메라**: 인물 손바닥 바로 위 (1.5–2 m). 인물 얼굴 최대 1/3 만 보이거나 안 보임. *손바닥 자체가 화면의 30–40%*.
- **광원**: 새벽광이 손바닥 *측면 낮은 각도* 로 닿음. 손금 잔주름이 광원 결대로 *옅은 강* 처럼 보이는 순간 — *해가 들기 직전 / 들기 시작* 의 1 분 자리.
- **자국빛**: 손금 잔주름 안쪽에 자국빛 (`#C8995A` 패밀리) 가 *옅게 잠시* 비침. 면적 ≤ 0.5% — *자국빛 1–3% 룰의 절반 이하*. *축복의 결을 떠벌림 트립* 임계에서 안전.
- **금지**: 손바닥 클로즈업 정면 (얼굴 인식 위반 위험), 손금이 *지도 도판처럼* 그려짐 (실제 등고선·강·지명 라벨 형태 — §3 *완성된 지도 도판* 위반), 손바닥 위에 *완성된 자기 모습* 이 비침 (반사·홀로그램 — §3 SF 트립와이어).
- **신호**: *햇빛이 들면 그림이 사라진다* (bible §4) — 컷의 *직후 1 컷* 이 *햇빛이 든 손바닥 = 잔주름만* 으로 따라오는 시퀀스 권고. 단일 컷에서 처방 시 *손금이 옅게 흐려지는 그라디언트* 로 *사라지기 직전* 결을 박을 것.

> 본 컷이 시각이 직접 박는 *자립* 의 1 차 그림. *누구도 대신 펴 줄 수 없는 손바닥* 이 *시각으로* 입증된다.

#### 5.4.2 *쉼의 자리* 컷 (bible v0.2 §3)

본 컷의 1 차 시각 정의:

- **카메라**: 인물 무릎 높이, 종이 바닥과 인물 양쪽이 모두 화면에 들어오는 *옅은 광각*. 인물은 *앉아 있거나 종이에 손 짚고 있음*. 화면 1/8~1/6 룰 (§3) 그대로.
- **자국 분포**: 인물 발치 주변에 *옅어지지 않은* 자국 1–3 점이 박혀 있음. 자국빛 0% (이 자리는 새 자국이 박히지 않으므로 광휘 부재). 단 자국 자체의 *진하기는 평균보다 살짝 진함* — *고요한 보존* 의 시각.
- **시간 결**: 새벽광이 *인물 그림자를 옅게 늘어뜨림* — 멈춰 있어도 *살아 있는* 결. 광원은 §4 그대로 낮은 수평선.
- **광원**: 새벽광이 인물보다 *종이 쪽으로 살짝 더 머묾*. *종이가 식지 않는다* (bible §3) 의 시각 박음.
- **금지**: 인물이 *완전 정지* (걸음 직전/직후 결 부재) — 본 v0.1 §3 *완전 정지 자세 금지* 위반 위험. *쉼의 자리* 라 해도 *직전에 막 앉음 / 직후에 다시 일어나려 함* 의 결을 한 자리 박을 것 (예: 한 손이 종이를 짚었으나 손가락 끝이 살짝 떠 있음).
- **신호**: 컷에 *나림* 의 12 색조 박물관 (character-sheets §나림) 처럼 *남의 hex 색조 흙* 이 1–3 점 박혀도 좋음 — 그러나 본 컷의 1 차 결은 *멈춤 ≠ 처벌* 이지 *연대* 가 아니므로, 색조 다양성은 부속 결.

> 본 컷이 시각이 직접 박는 *축복* 의 2 차 그림 (1 차는 §6 *빈 종이 면적 ≥ 60%*). *멈춰 있어도 종이가 식지 않는다* 가 *시각으로* 입증된다.

#### 5.4.3 *화살표 자국* 컷 (bible v0.2 §5)

본 컷의 1 차 시각 정의:

- **카메라**: 인물 무릎 높이 또는 자국 *바로 위 (top-down 아님 — 자국 옆에서 비스듬히)*. 자국 1–3 점이 화면 1/3 이상 차지하는 *자국 클로즈업*. *부감 / 신적 시점* 위반 임계 근접 — 카메라 각도 35–55° 사이만 (top-down = 위반).
- **자국 결**: 자국마다 *발끝 방향* 이 보임. 흙 결의 한쪽 가장자리가 *화살표 끝처럼 살짝 뾰족*. v0.1 의 *자국 = 점* 룰을 *자국 = 작은 화살표* 로 갱신.
- **방향 분포**: 컷에 자국 ≥ 3 점인 경우 *방향 분포* 가 즉답 가능해야 함 — 한 방향 모임 (작정한 사람) / 흩어짐 (헤맨 사람) / 두 방향 갈라짐 (큰 결단을 한 사람) 셋 중 하나로 *5 분 cold reader* 가 즉답.
- **광원**: §4 그대로. 단 자국 클로즈업이라 *자국빛이 발끝 안쪽 가장자리* 에 박힘 — *진심으로 걸은 자국에만* 의 결.
- **금지**: 자국이 *지도 등고선처럼* 그려짐 (forbidden-language §3 *완성된 지도* 시각화 위반), 자국이 *너무 많아 군중 결* (§5.2 *3 인 이상 군중 금지* 위반 임계 — 자국 ≤ 7 점 / 컷 권고).
- **신호**: 본 컷의 *방향 분포* 가 곧 인물의 *의지의 모양* (bible §5). 인물이 컷에 같이 등장하면 *인물의 시선이 자국 방향과 한 결* — 다른 결이면 *내적 갈등* 결로 의도 사용 (예: 정해의 *발끝 방향만 단정* 결).

> 본 컷이 시각이 직접 박는 *방향* 의 1 차 그림. *방향 분포 = 의지의 모양* 이 *시각으로* 입증된다.

### 12.1 강 / 도시 / 바다 (terrain-v0 §강·도시·바다 시각 룰)

- **강** = *옅은 자국이 줄지어 굳은 자리*. 시각: 자국 hex 가 *한 줄로* 박힘 + *물의 광택 / 반사 금지* — *흐르는 물 시각* 위반 (forbidden-language §흐름 결). *흐름* 은 *방향 일치한 자국* 의 시각으로만.
- **도시** = *통과의 산 위에 일의 산 빽빽이*. 시각: 4 톤 hex 혼재 + 정사각형 / 직선 구조물 *전혀 없음* (forbidden-language §완성된 지도 시각 위반). *건축 형태 0* — 도시도 *흙 자국* 결로만.
- **바다** = *없음*. 시각 자체에 등장 금지 — 종이 가장자리 너머는 *옅은 새벽광 그라디언트* 로 처리. *수평선 = 다른 사람의 종이 시작* 이라는 의미를 *시각으로* 박지 않음 (메타 노출 회피) — 단순히 *백지로 녹는 가장자리*.

## 15 *세 축 부재* 임계 응답 (NEW — voice-keeper r2 §C.2 직접 응답)

> voice-keeper r2 §C.2 *세 축 부재* 임계 발화 — cy-001 누적 *나* 축 5 자리. 본 §은 art-director 가 *시각 측에서 할 수 있는 것 / 할 수 없는 것* 을 박는 자리.

### 15.1 시각이 *할 수 있는* 것

- **인물 hex 6 자리 중 3 자리 보존** (본 v0.2 §11.5) — *가족 축 1 + 업 축 1 + 동행자 미정 1*. writer r3 또는 loremaster r3 의 *가족 / 업 축 인물 시트* 도착 시 본 보존 자리에 hex 박음. 시각이 *기다리는 자리* 를 미리 박음.
- **§5.4 신설 컷 3 종이 *나* 축 1 인칭에 편향되지 않게** 박힘 — *손금 새벽* (해온 1 인칭) 외 *쉼의 자리* (나림 1 인칭) + *화살표 자국* (정해 1 인칭). 향후 *가족 축* (예: 부모와 아이가 *겹친 자국* 1 점에서 만나는 컷) / *업 축* (예: 일의 산 위에서 *지도-스승* 1 인이 옅은 가장자리를 보는 컷) 의 시그니처 컷 후보 신설 가능.
- **terrain-v0 5 산 외형 룰 (§12) 자체가 *축* 무관** — 사랑의 산 / 일의 산 / 후회의 산 / 기다림의 산 / 통과의 산 모두 *나·가족·업* 어느 축에서도 활용 가능. *가족 축의 기다림의 산* / *업 축의 일의 산* 등 결합 자리 박음.

### 15.2 시각이 *할 수 없는* 것

- **축 자체는 인물 행동·관계의 자리** — writer / loremaster 의 산출이 *1 차 진실*. art-director 는 *시각 후속*.
- 시각이 *가족 축 인물 외형* 을 단독으로 박을 수 없음 — *가족 축 인물 시트 (행동 / 관계 / 동기)* 가 도착해야 외형 hex 가 의미를 가짐. 본 v0.2 §11.5 *보존 1 자리* 가 *대기* 자리.
- 시각이 *세 축 균형* 을 *측정* 할 수 없음 — voice-keeper 의 정렬 채점이 *측정 도구*. art-director 는 *측정에 응답한 시각 보강* 만.
