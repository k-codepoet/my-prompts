# 2D Pixel Art UI Panels (9-Slice / Border-Image)

Generated on: 2026-04-18  
Style: Retro pixel art, 9-slice border-image style, RPG game HUD  
Resolution: 512x512  
Workflow: z-image-turbo (ComfyUI)  
Usage: `border-image` CSS property 또는 게임 엔진의 9-slice UI 패널로 사용

---

## 9-Slice 사용법

```css
.panel {
  border-image: url("panel.png") 64 fill / 64px / 0px stretch;
  /* 슬라이스 오프셋은 실제 이미지의 코너 크기에 맞게 조정 */
}
```
