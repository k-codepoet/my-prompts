## §3. terrain v1 산출물 골자 (요약)

8 종 사람 사는 지형 1 차 축:

1. **산골** — 비탈 + 좁은 길 + 한 줄 마을 + 새벽 안개 + 가족 단위 + 천천한 박자
2. **들녘** — 평지 + 분산 마을 + 사방 바람 + 짧은 새벽 + 길고 균질한 박자
3. **강가** — 강 따라 한 줄 마을 + 옅은 자국 + 습기 + 동행 결
4. **포구** — 가장자리 + 떠남·돌아옴 + 짠 흙 + 받음 + 보냄 결
5. **구릉** — 곡선 + 넓고 낮은 마을 + 균질 + 부드러운 박자
6. **분지** — 산에 둘러싸임 + 정착 강 + 늦은 새벽 + 두꺼운 흙
7. **도시 변두리** — 일의 산과 들녘 만남 + 옮겨 온 결 + 안과 밖 박자
8. **도시 한복판** — 통과의 산 + 빽빽 + 익명 흙 + 가장 빠른 박자
